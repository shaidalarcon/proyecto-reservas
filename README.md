# 🏀 Proyecto FullStack - TOTSPORT

## Descripción
**TOTSPORT** es una aplicación FullStack desarrollada con el stack MERN (MongoDB, Express, React, Node.js) que permite la gestión integral de reservas en un complejo deportivo.

Este repositorio funciona como un **Monorepo**, conteniendo tanto el código del cliente (Frontend) como el del servidor (Backend) en una sola estructura unificada.

## 🚀 Enlaces de Despliegue (Live Demo)
La aplicación se encuentra completamente desplegada y funcional en los siguientes enlaces:
- **Frontend (Vercel):** https://proyecto-reservas-snowy.vercel.app/
- **Backend API (Render):** https://totsport-backend.onrender.com

## 🛠️ Tecnologías Globales
- **Frontend:** React, Vite, Chakra UI, React Hook Form, Axios.
- **Backend:** Node.js, Express, MongoDB, Mongoose, JWT, Cloudinary, Multer.
- **Despliegue:** Vercel (Cliente), Render (Servidor), MongoDB Atlas (Base de Datos).
